# Generated Project (aisci)

        **아이디어:** 구현하고자 하는 대상의 영감은 사실 트랜스포머와 아이언맨으로부터 얻었다. 그 중 트랜스포머의 진화가 더 오래 진행된 사례이고, 아이언맨은 개인이어서 타노스도 외로운 싸움을 했다고 칭찬을 한다. 그래서 오늘 구현할 것은 메타러닝이 반영된 줄기세포 배양체 안드로이드 및 휴머노이드이다. 줄기세포로 사람의 몸이 형태를 만들 수 있는 공간을 마련한 뒤에 그 공간을 감싸고 있거나 공유가 되어있는 물질은 사실 그래핀 폴리머이고, 이것이 전체 신체 구조를 구성하고, 그것이 트랜스포머와 같은 형태로 메타러닝을 통해서 사용자의 지도학습에 따라서 무엇이든지 다 될 수 있어야 하는 것이다. 이게 오늘의 목표이다.
        **목표:** 해당 프로젝트의 타이틀은 mtld 이고, 이 타이틀에 설계 반영 프로그램인 ParaView 6.0.1 의 특성을 최대한 반영하는 것이 이번 프로젝트의 목표이다. 본인이 생각하는 이상신소재는 그래핀 폴리머 액정으로 이 소재의 특성을 분명히 ParaView 가 잘 알고 있겠지만, 해당 프로젝트는 소재를 구현하는 프로젝트가 아니라 줄기세포 배양체를 구현하는 프로젝트이다. 여기에 더해서 Paraview 에 신소재공학 및 생명공학이 반영되지는 않을 것이고, 대신에 줄기세포 배양체의 물리적 그리고 공학적 설계 만을 담당할 것이다. 쉽게 말하면 줄기세포를 배양할 수 있는 안드로이드 아니면 휴머노이드를 구현하는 것이 본 프로젝트의 의의이다.
        **제약:** ParaView 의 특성을 최대한 반영하고 메타러닝의 자각 능력을 반영시키기 가장좋은 환경은 파이참 환경이고, 해당 환경에서 적절한 파이썬 환경이 잘 반영되면 된다. 그래서 파이썬 버전을 알아놔야 하는 것이다. 재미있게도 파이썬 기능은 사실 윈도우 주소를 통해서 알 수도 있다. 그래서 파이참을 쓰려는 것이고, 이어서 ParaView 에서 작업했다면 이 환경에 맞는 줄기세포 배양체로 출시될 안드로이드나 휴머노이드를 다시 구현하려고 하는 것이다. 그래서 필요한 환경은 파이참, ParaView, 그리고 메타러닝이란 기술이다. 단, 설계에 반영하지 않을 항목들은 생명공학 및 소재공학 관련 분야이다. 오직 인체공학 뿐이다.
        **기술스택:** Flask, SQLite, JS
